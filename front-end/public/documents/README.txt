Place your PDF files in this folder with these exact filenames:

  annual-report-2025.pdf       → Latest Audited Financials
  annual-report-2023.pdf       → 2023 Annual Report
  fcra-return-2024.pdf         → FCRA Return (2024)
  12a-80g-certificates.pdf     → 12A & 80G Certificates
  utilisation-certificate-2024.pdf → Utilisation Certificate 2024
  trust-registration.pdf       → Trust Registration Certificate

Once files are placed here, all download buttons on the Financials page will work automatically.
